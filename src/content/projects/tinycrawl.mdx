A roguelike with a hard constraint: the entire game — engine, art, audio cues,
save system — ships as one `index.html` you can email to a friend. No build
step, no dependencies, no server.

The constraint forced every interesting decision. Procedural dungeons because I
couldn't ship hand-authored levels in the byte budget. Permadeath because save
files would have meant local storage plumbing I didn't want. The "weirdly
emotional ending" was an accident of the constraint too — there was only room
for one, so it had to land.
