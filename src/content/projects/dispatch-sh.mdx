I take notes in a dozen places and read them in none. `dispatch` walks the
folders I actually use, pulls anything touched in the last seven days, and
assembles a single markdown digest I can read on Sunday with coffee.

It is the smallest possible thing that solves a real annoyance, which made it
the perfect excuse to learn Rust properly — error handling, the borrow checker,
shipping a real binary — without the project itself mattering enough to stall on
architecture.
